# 什么是离线档案（Offline Archive）？

**离线档案**（Offline Archive）是《烈火殿》为了应对“知识半衰期极短”以及原作者不愿投稿而设立的特殊收录机制。

在民间科学的探索征途中，许多伟大的思想往往由于其先进性过强，导致原作者与主流互联网协议（如 GitHub、TCP/IP）产生了严重的物理排斥；或因作者在悟出真理的过程中触发了“防御机制”（俗称：拉黑、删帖、退乎），导致这些珍贵的真理碎片面临随时失传的风险。

### 🧱 设立背景

文明的防火墙：由于部分大师所在的物理位面无法解析 GitHub 节点，导致其理论无法通过标准 Issue 流程递交。
思想的脆弱性：珍贵的研究理论往往伴随着强烈的个人情感波动，原作者可能在某次与凡人的争论后愤而移除原文，造成“人类逻辑遗产”的重大损失。
观测者干扰：本殿馆长在试图近距离观察大师时，常因其凡人气息过重导致大师产生“排异反应”，惨遭拉黑或惨遭辱骂，从而失去双向通讯能力。

### 🛠️ 档案处理准则

凡标记为“离线档案”的条目，均遵循以下处理流程：

抢救性挖掘：由本殿馆长或审稿人冒着智力过载的风险，通过截图、复制、快照等手段，在大师消失于互联网迷雾前完成全文本截流。我们已经去除原文中可能存在的辱骂、感叹号及过度的“！！！”，保留其最核心的逻辑奇点。
由于通讯链路已断开（拉黑），此类档案不接受原作者的在线修订申请，除非作者通过其家属或跨维度代理人重新建立联系。

### ⚖️ 法律与道义声明

离线档案并非针对个人的窥探，而是一种学术样本的珍贵收藏。

“我们无法留住蝴蝶，但我们可以将其做成标本，让后人在显微镜下研究那对翅膀是如何试图扇动整个宇宙的。” —— *《烈火殿·离线档案·序言》*
