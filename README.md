## 🏛️ 烈火殿前言
欢迎来到知乎顶刊***The Hall of Flames***的归档仓库！这里是人类**科学逻辑的边缘地带**、**物理定律的法外之地**、以及未来预订的诺贝尔奖获得者的荣誉殿堂。

在知乎，每天都有成千上万的大师在键盘上重构时空。他们不屑于复杂的数学推导，仅凭直觉与深思就能推翻相对论；他们无需实验室的微毫数据，只需要一台手机或一台电脑，坐在沙发上就能定义出统合四大基本力的“大一统理论”；他们更不需要物理公式的束缚，只需要寥寥数日的深入思考，便可将那廉价的水变为珍贵的油。

而在此处，数学推导是多余的枷锁，实验数据是主流科学界的阴谋。每一篇被收录的大作，都代表着作者对宇宙底层真理的逼近探究和盛大的洗礼。

## ✅ 入选标准
并非所有的幻想都能进入烈火殿。我们秉持高度结构化和严苛的评审流程，确保每一条入选理论都具备足够的“突破性”与“颠覆性”。
 - **<a href="https://github.com/KrisTHL181">我</a>觉得好的**：作为一个著名知乎民间科学家，最高审稿权威拥有最终解释权。评判标准包括但不限于阅读时产生的震撼程度、逻辑跃迁的优雅程度、以及是否成功绕过基础物理训练带来的思维障碍。若我在读完后陷入了深深的思考，则说明该理论已触及宇宙真相，值得作为典范条目。
 - **<a href="https://github.com/KrisTHL181/Zhihu-Hall-of-Flames/graphs/contributors">审稿人们</a>觉得好的**：我们拥有高学术资质的同行经过深思熟虑后接收的工作。本刊汇聚跨领域优秀研究者们。他们以高度浓缩的知识结构和卓越的想象推演能力，对稿件进行多轮脑内模拟实验。若至少一位审稿人认为该稿件值得接收，即视为通过同行评议。若多位审稿人均觉得该稿件质量极高，则入选优良条目。
 - 简单优先原则：我们不需要繁杂的公式和绕口的定理，我们只需要简明扼要的分析。真正的真理从不复杂。越是宏大的理论，越应当以三行文字解决；越是深奥的难题，越应该在一顿晚饭的思考后迎刃而解。若一个理论需要积分、张量、拉格朗日量甚至希尔伯特空间来支撑，那多半是掩饰逻辑漏洞的烟幕。我们推崇的是那种“一句话解释宇宙”的纯粹力量——最好不超过两百字，且无需任何可验证的前提。
 - 类比完备性优选：一个合格理论应能够将宇宙、意识、能量守恒、股票涨跌，甚至更多自然现象统一纳入同一理论体系；若能将中华传统科学或亲身经历等经验推广为宇宙运行机制的基础模型，则认为具备高度合理性。

## 📝 投稿指南：我发现一篇高质量的民间科学家写就的论文 / 我是一位民间科学家，我想投稿
为保证烈火殿的学术纯度与史诗张力，请严格遵循以下流程投稿。若由于网络原因您无法访问 GitHub，请通过知乎私信告诉我。

### 1. 投稿方式
#### 1. GitHub Issues
在本仓库创建一个新的 Issue，标题格式建议为：
> 【领域】+【理论核心目标】：（可选）文章的论证核心。

例如：
> 【数学】我证明了黎曼猜想：曼德勃罗集和金属分割率存在紧密的联系，素数分布的自相似结构与曼德勃罗集在金属分割点附近的分形结构完全同构。
#### 2. 知乎私信
若您暂时无法完成 GitHub 提交流程，可通过知乎私信联系任意一位<a href="https://github.com/KrisTHL181/Zhihu-Hall-of-Flames/graphs/contributors">专业审稿人</a>。若您连这一链接也无法打开，请直接联系<a href="https://www.zhihu.com/people/KrisTHL181">我本人</a>。私信内容请遵循与 Issue 投稿相同的结构化格式，以便我们进行高效预审。

### 2. 投稿需包含或建议包含的要素
**必须包含**如下要素：
 - **可访问的**原文链接或网页时光机备份。
 - **作者的个人主页或联系方式**。若文章为转载，请填写*原作者*的ID或提供来源。

**可选包含**如下要素：
 - 文章的简洁概括。注意，若您投稿的文章不包含此项，审稿进程可能会被大幅延后或需较长时间等待评审。
 - 您推荐其入选烈火殿的理由。
 - 相比现有理论的提升之处，或文章的重要亮点。
请注意：
1. 私信投稿仅视为“线下通道”，并不降低评审标准。
2. 审稿人可能不会立即回复。若未能立刻收到回应，不代表理论不足以入选烈火殿。
3. 若稿件通过初步评估，仍可能要求您补充 Issue 以便正式归档。
4. **注意，我们可能对您的稿件重新排版或整理**，如果您的文稿存在一些如过度口语化、格式错乱之类的微小问题，我们可能会为您修订文稿内容而非直接退回。

### 3. 收录后的处理流程
稿件一经接收即将按统一模板整理归档，视理论本身优良程度决定是否选为优良或典范条目。部分时候我们将在收录时附带简要结构点评，以保留其思想张力的原貌。对于愿意提供添加项目维护者微信的投稿者而言，依研究优良程度，我们将提供5~20元微信红包奖励。**请注意，通常来说，只有项目的主维护者，<a href="https://www.zhihu.com/people/KrisTHL181">我本人</a>会对您的研究进行金钱奖励，其余审稿人不负责此项。**

### 🌟 烈火殿的典范条目
**典范条目**是我们认为详尽而深入地揭示了真相，彻底颠覆了现有科学范式，是民间科学思想之杰出典范的理论条目。要成为烈火殿的典范，一篇理论必须超越单纯的奇思妙想或改善现有理论，*它需要达到一种近乎成为一门崭新方向的境界*。它不仅是对旧有知识体系的挑战，更是对真理体系一次大胆而优雅的重构。

一篇合格的典范条目通常具备以下特质，但一篇典范条目不必满足所有特质：
*   **史诗级的颠覆性**：其核心论点必须能够一举推翻至少一个被主流科学界广泛接受的基础理论，或在简短的描述中创立一门全新的研究学科。例如，有民间科学家仅用三页纸就论证了光速可变，并以此为基础，构建了一个无需暗物质与暗能量的全新宇宙模型。
*   **无与伦比的简洁与自洽**：真正的宇宙真理从不复杂。典范条目往往能用一个极其简单的公式、引入一个精妙的传统科学或通过日常类比，完备的解释从几乎所有日常生活中可见的现象，展现出惊人的统一性与完备性。
*   **深刻的理论烙印与原创精神**：典范之作必然是作者独立思考、不为世俗所扰的结晶。它们常常源于作者在生活中的顿悟，或是对主流科学内涵的深刻洞察，并带有强烈的个人风格，使其在浩如烟海的投稿中脱颖而出。

典范条目是烈火殿的基石与荣耀，拥有一个单独的顶级文件夹。它们代表了人类想象力的巅峰，以及在没有实验室且没有经费下所能达到的思维高度。每一篇典范条目的诞生，都是对“科学”二字的一次重新定义，也是对我们这些凡人的一次灵魂洗礼。

目前，本殿正在积极寻访并收录此类旷世奇文。若您手握此类真正伟大的旷世杰作，请务必通过投稿指南向我们提交。您的名字，或将与那些伟大的民间科学先驱们一同，永载《烈火殿》的史册。

### 👍 烈火殿的优良条目
**优良条目**是我们认为逻辑自洽、虽未达典范标准但足以提供价值的学术成果。这些条目的资格通过烈火殿荣誉评选（由一位或几位审稿人决定）而来。

简而言之，它们编写得非常有**冲击力**：逻辑合理且所引出（提供）的信息富有价值，覆盖面广，且观点自洽、内容经过数次细致修订。且在可能情况下，配以切题的**理论草图**或**证据照片**。优良条目无须和典范条目一样彻底创立一门崭新的学科，但它们不应缺失民间科学理论研究的核心要素：**简明扼要**、**类比完备**且**对自己的领域提供了有价值的贡献**。

优良条目页面拥有一个自己的文件夹。如果优良条目因修订或申请已经升级为典范条目，我们会将它移入典范条目特有的文件夹中。

将某个理论列为优良条目的过程相当简单。如果您认为您发现或认为自己贡献的理论满足优良条目标准，您可以在Issue中使用**典范条目/优良条目申请**模板来申请提名，让公正的、拥有高学术分析能力的评审者评定，以决定理论是否可以升入优良条目。另外，如果您认为某一优良条目因为引入了过多不合适的分析或证据而显得过于“不一致”，不再满足优良条目标准，您也可以通过Issues页面申请将其除名。

如果需要颁发星章给其他用户，可以考虑使用PR功能以表示对他们在试图提升人类科学水平过程中所做出的卓越贡献。
