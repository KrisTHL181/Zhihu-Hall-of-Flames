# 📚 普通文献 | General Docs

> **“不是所有真理都需要颠覆世界，有些只是安静地躺在角落，等待被重新发现。”**

## 📜 区域定义

**普通文献**（General Docs）是《烈火殿》中最为包容的收藏区域，专门收录那些**虽未达到“典范”或“优良”标准，但仍具有一定思想价值或保存意义的民间科学论述**。

在民间科学研究的生态系统中，并非每一篇文章都能成为撼动物理学大厦的史诗级巨作。有些理论或许逻辑不够严密，有些观点或许类比不够完备，有些思考或许只是灵光一闪的碎片——但它们同样是民间科学思想的重要组成部分，是大师们探索真理过程中的真实足迹。

本区域不做严苛的学术筛选，不设复杂的评审流程。我们秉持的基本原则是：**只要它体现了一定程度的独立思考，只要它展现了作者对科学问题的真诚探索，它就值得被保存。**

## 🧾 收录范围

凡符合以下任一特征的文献，均可归入本区域：

### 1. 萌芽期理论

那些尚不成熟、但已显示出独特思考方向的理论雏形。可能是作者在评论区的一次精彩回复，也可能是想法区的一段闪念记录。这些文献如同思想的种子，虽然尚未长成参天大树，却蕴含着未来突破的可能性。

### 2. 单一洞见型文献

全文只有一个核心观点，但这一观点足够有趣。例如：“我突然想到，时间可能只是熵增在意识中的投影”——仅此一句，别无长物。虽不足以构成完整理论，却足以引发读者的思考。

### 3. 边缘交叉探索

涉及多学科交叉但论证不够系统的尝试。比如用经济学原理解释量子坍缩，或用音乐理论重构化学元素周期表。这类文献往往充满想象力，但缺乏足够的学科支撑。

### 4. 学术考古发现

那些发表于多年前、如今已被遗忘的民间科学文章。它们可能是某位早期知乎用户对“永动机”的天才构想，也可能是某位博客作者对“反引力”的系统论证。虽已过时，但作为思想史的一部分，仍具保存价值。

### 5. 翻译引进文献

从其他平台或外网挖掘的、尚未在国内传播的优秀民间科学作品。语言不应成为思想传播的障碍，本区域欢迎各类跨越语言边界的科学奇文。

### 6. 无法归类的边缘案例

那些难以被“馆长提名”、“离线档案”、“连环论文”等专区接纳，但又确实具有一定可读性的作品。这里是它们的避风港。

## ⚖️ 收录原则

### 无需审核
与“典范条目”和“优良条目”不同，普通文献的收录无需经过同行评议。任何人均可通过标准流程提交，只要内容符合“民间科学”的基本定义，即可归档。

### 保持原貌
我们尽可能保留文献的原始形态，包括其语言风格、论证方式甚至错别字。如需对格式进行微调以便阅读，会在文献开头注明“**本版本经轻微格式整理，内容未作任何修改**”。

### 分类标注
每篇普通文献在归档时，将标注其所属类型（如“萌芽期理论”、“单一洞见”、“学术考古”等），以便读者快速识别。

### 动态升级
普通文献并非永久定级。若某篇文献在后续修订中逻辑日趋完善，或因其前瞻性被后人重新发现，可经评审流程申请升级为“优良条目”甚至“典范条目”。

## 🔗 与其他区域的关系

| 区域 | 关系说明 |
|------|----------|
| **馆长提名** | 馆长提名为“一票入馆”的特权区域，收录的是让馆长“物理眩晕”的神作；普通文献则是让馆长“点头称是”的凡品。 |
| **离线档案** | 离线档案侧重于“抢救性挖掘”，收录的是面临失传风险的珍贵文本；普通文献则更多是常规性收集。 |
| **连环论文** | 连环论文是连载中的思想史诗，强调多篇文章的逻辑递进；普通文献多为单篇独立论述。 |
| **数据存储** | 数据存储保存理论的原始证据形态（图片、压缩包等）；普通文献保存的是文本本身。 |

## 📝 提交指南

若您发现一篇值得保存但不够“炸裂”的民间科学文章，欢迎通过以下方式提交：

1. **GitHub Issues**：创建新 Issue，标题格式为“【普通文献】+文章标题”，内容附上原文链接或截图。
2. **知乎私信**：联系任意一位仓库维护者，提供文章链接及简要推荐理由。

提交时请注明：
- 原文作者及可追溯的来源链接
- 文章的发布日期（如已知）
- 您认为其值得收录的简要理由
